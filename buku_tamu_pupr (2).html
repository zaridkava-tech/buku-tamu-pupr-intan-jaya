Integrasi Google Apps Script (SITAMU)

Ikuti panduan berikut agar aplikasi ini terhubung ke Google Sheets Anda secara gratis.

Siap Hubungkan

Langkah 1: Buat Spreadsheet baru di akun Google Drive Anda, beri nama Buku Tamu PUPR.

Langkah 2: Buat struktur kolom di Sheet 1 bernama DATA_TAMU pada baris pertama:

ID | Tanggal | Jam | Nama | Instansi | No HP | Alamat | Bidang | Pegawai Tujuan | Keperluan

Langkah 3: Pada menu Spreadsheet, buka Ekstensi > Apps Script.

Langkah 4: Hapus file Kode.gs bawaan, ganti dengan salinan kode backend di bawah ini:

// ==================== SALIN KODE BERIKUT KE KODE.GS ====================
function doGet() {
  return HtmlService.createTemplateFromFile('Index')
      .evaluate()
      .setTitle('SITAMU - Dinas PUPR Intan Jaya')
      .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL)
      .addMetaTag('viewport', 'width=device-width, initial-scale=1');
}

function simpanTamuSheets(data) {
  try {
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = ss.getSheetByName("DATA_TAMU");
    if (!sheet) {
      sheet = ss.insertSheet("DATA_TAMU");
      sheet.appendRow(["ID", "Tanggal", "Jam", "Nama", "Instansi", "No HP", "Alamat", "Bidang", "Pegawai Tujuan", "Keperluan"]);
    }
    
    var tgl = new Date();
    var idUnik = "TAMU-" + tgl.getTime();
    var tglString = tgl.toLocaleDateString("id-ID");
    var jamString = tgl.toLocaleTimeString("id-ID", {hour: '2-digit', minute:'2-digit'});
    
    sheet.appendRow([
      idUnik,
      tglString,
      jamString,
      data.nama,
      data.instansi,
      data.nohp,
      data.alamat || "-",
      data.bidang,
      data.pegawai,
      data.tujuan
    ]);
    return "SUKSES";
  } catch(e) {
    return "GAGAL: " + e.toString();
  }
}

Langkah 5: Di editor Google Apps Script, buat file HTML baru beri nama Index.html, tempelkan seluruh isi file HTML ini (file tempat Anda melihat halaman ini) di sana.

Langkah 6: Deploy sebagai Aplikasi Web (Web App). Set "Execute as" sebagai akun Anda, dan "Who has access" sebagai Anyone. Selesai!
